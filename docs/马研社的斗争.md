# **马研社的斗争**

&emsp;&emsp;一年时间里，在小资产阶级、右倾主义者、教条主义者的层层包围中，产生了一个红色的左倾的社会性组织，在应试、欺诈、垄断的社会环境下，组织成员正在不断发展，这些人来自马研社，他们都是青年人他们会用思想与实际行动武装自己，从而在这肮脏的环境中作出斗争与反抗！

&emsp;&emsp;这个组织，无论在目前还是就长期来看，都是以学术讨论为中心的组织，而非什么政治性组织。组织成立的初心就是对于带有小资思想的教育工作者的压迫行为的斗争以及反对修正主义性质的行为。而为了实现我们的初心，我们还需要具备下列的条件：

!>（1）有很好的群众；<br>
（2）有很好的组织；<br>
（3）组织成员普遍具有良好的思想素养；<br>
（4）组织成员对于组织工作的向心力；<br>
（5）有便于斗争的优势。

&emsp;&emsp;组织的中心机构：组织部、宣传部。组织部致力于协调各分部的关系并统合分部以进行思想宣传，同时工作于提高组织内成员素质、向外部联系等事项；宣传部致力于宣传工作，宣传的形式包括但不限于为宣传部写稿件投稿至《民潮》和《思复》杂志、制作视频等。其中关于《民潮》及《思复》杂志线下印刷由各分部筹集钱款，宣传部不负责进行线下杂志打印工作，仅负责杂志稿件处理。

&emsp;&emsp;组织的中心工作：进行宣传，同时扩大规模以更好宣传。

&emsp;&emsp;组织领导人：组织内施行民主政策，无实际意义上的领导人，设有精神领导者（吉祥物），实际权力掌握在组织部委员会中，任何组织成员都有义务维护组织部委员会的权力不受个人掌控。

&emsp;&emsp;组织工作的开展：组织成员大多步入高中学习阶段，为了日后组织成员继续开展思想宣传工作并同修正主义作斗争，组织中心决定在各优质普通高中建立起马研社分部，并在分部确立一名到两名甚至更多负责人进行领导与组织分部的工作；同时继续出版及刊印《民潮》和《思复》两刊物作为组织报刊起思想宣传作用，并要求各分部募捐自购至少一本《共产党宣言》用以提升分部成员理论水平；组织内提倡具有各种不同的思想的成员（右倾思想、机会主义潮流、纳粹主义等除外）进行学术讨论，主流思想仍应为马克思列宁主义，允许对马克思列宁主义有一定程度的反对及批判，欢迎异见者加入讨论；马研社希望能同其他志同道合的左派组织进行思想宣传并团结一切可以团结的对象，搜集其他组织的情报，联合其他左派组织一同宣传；马研社需要左翼思想者、严格保守组织秘密者、对社会现状清醒认知者、敏于思考且有思想基础者，壮大组织有生力量，更强有力地进行斗争！

&emsp;&emsp;组织部委员会：该委员会为组织的权力机关，其原则遵循列宁“一切权力归于苏维埃”。由各分部负责人组成马研社组织部委员会，各负责人作为委员每月进行对于分部的管理宣传经验的讨论，各委员有权提出各项议案以促进组织发展、影响组织未来。委员有权监督组织内成员遵守委员会规定并保持秘密。每一名委员在地位上同组织成员毫无区别，仅仅是能力区别，不容许组织内的任何人身攻击。

&emsp;&emsp;分部要务及协作：视各分部具体情况每月提供《民潮》文章类稿件一至二篇，《思复》文章类稿件一至二篇，诗词类稿件则一次投稿五篇以上。各个分部之间无论大小，地位相平，各分部间应持友好互助原则行事，在宣传工作中互相协调，携手并进。

&emsp;&emsp;组织成员的地位与职责：每一名马克思主义研究社的成员在任何时候皆同他人平等，能力除外。每一名组织成员都有权向委员会提出议案，成员应严格保守组织秘密，维护组织部委员会的权力不受个人控制。要自发的学习、斗争，联系其他组织，在组织内团结、紧张、严肃、活泼。

&emsp;&emsp;组织给予每位组织成员发言的权利，同时力求让每位组织成员学到更多，能够拓宽自己的眼界为社会着想，为社会主义着想。马研社不会容许对其成员的任何污蔑和诋毁，无论个人或组织。

&emsp;&emsp;写在最后：从前的革命的都结束了，新的斗争开始了！同志们，让我们团结起来，为了目标的实现，不断地学习与斗争！
